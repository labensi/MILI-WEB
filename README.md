# MILI CHAT 🌸

Your cute little corner of the internet - a social media & chat web application with Discord-style rooms, Instagram-like feed, and adorable aesthetics.

## Features

- 🎨 **Beautiful Glassmorphism UI** with seasonal themes
- 💬 **Real-time Chat Rooms** with Discord-like features
- 📸 **Social Feed** with image/video/text posts
- 💌 **Direct Messages** with full chat features
- 👤 **Rich User Profiles** with avatar frames and effects
- 🎭 **Custom Font Styles** for usernames
- 👑 **Admin Panel** for moderation
- 🔔 **Real-time Notifications**
- 🎪 **Stickers & Custom Emoji**
- 🎵 **Ambient Sound** per season

## Tech Stack

- React 18 + Vite + TypeScript
- TailwindCSS + Framer Motion
- Firebase (Auth, Firestore, Storage, Realtime DB)
- Zustand for state management
- React Router v6
